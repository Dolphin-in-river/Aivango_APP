# iVenGo — Платформа автоматизации рыцарских турниров

**iVenGo** — современная веб-система для организации и проведения рыцарских турниров в духе исторического романтизма.  
Проект полностью автоматизирует процессы от создания мероприятия до распределения призов и формирования отчётов.

## Основные роли пользователей

- **Организатор** — создаёт турниры, управляет заявками, сеткой, финансами и отчётами
- **Рыцарь** — подаёт заявки, отслеживает статус, участвует в боях
- **Спонсор** — вносит средства, выбирает пакеты (Bronze–Platinum), получает рекламные привилегии
- **Зритель** — покупает билеты, голосует за Приз Зрительских Симпатий (ПЗС)

## Ключевые возможности

- Регистрация и вход по email + одноразовый код подтверждения
- Создание турнира: выбор типа сетки (олимпийская / круговая), даты, категорий, призового фонда
- Подача и рассмотрение заявок рыцарей (статусы: PENDING → APPROVED → REJECTED)
- Автоматическое формирование турнирной таблицы / сетки
- Система спонсорства: пакеты с логотипами, баннерами, упоминаниями
- Продажа билетов зрителям + генерация QR-кодов
- Голосование зрителей за Приз Зрительских Симпатий
- Автоматический расчёт призового фонда после завершения турнира  
  (50% — 1 место, 25% — 2 место, 10% — 3 место, 5% — ПЗС)
- Формирование и отправка отчётов на почту (финансовый, по посещаемости, по результатам боёв)
- Личные кабинеты для всех ролей с историей участия / спонсорства / посещений

## Технологический стек

**Backend**  
Java 17 · Spring Boot · PostgreSQL 14 · Spring Security (email-код) · JPA/Hibernate · JMeter

**Frontend**  
React 18 + TypeScript

**Инфраструктура**  
Docker + Docker Compose · GitLab CI/CD · Nginx (прокси)

## Запуск проекта

### 1. Backend
```bash
cd aivango_backend
mvn clean package
java -jar target/aivengo-backend.jar
```

### 2. Frontend
```bash
cd aivango_frontend
npm install
npm run dev
```

### 3. Откройте в браузере
http://localhost:5173
(по умолчанию backend на http://localhost:8080)
